Git

Git itu gratis dan merupakan aplikasi sumber terbuka (dimana sumber kodenya publik) ditulis oleh Linus Torvalds yang juga menulis inti dari sistem operasi Linux.

Git adalah sebuah program untuk memantau perubahan seiring waktu, dalam pemrograman disebut sebagai kontrol versi. Jika kamu sudah pernah memakai fitur pelacakan perubahan di aplikasi teks editor maka kamu sudah familiar dengan konsepnya!

Pasang Git

Kami menyarankan untuk memasang Git di komputer kamu dengan cara mengunduh aplikasi GitHub Desktop. Kita tidak akan menggunakan aplikasi desktopnya (kita akan belajar terminal!) tapi didalamnya sudah termasuk Git dan ini cara termudah untuk memasang Git pada semua platform dengan cara yang sama.

  • Windows: Gunakan Git Shell untuk terminal.
  • Mac: Buka GitHub Desktop dan dari Pengaturan, pilih pasang alat baris perintah. Gunakan aplikasi terminal sebagai terminal pilihanmu.

Sudah punya Git atau tidak yakin? Coba ketik git --version di terminal milikmu dan jika hasilnya berupa nomor versi yang lebih besar dengan 1.7.10, kamu sudah siap! Untuk informasi lanjut, kunjungi situs resmi Git.

Perangkat Lunak Git

Aplikasi Github Desktop dapat melakukan banyak hal dengan Git tapi tidak semua, itulah alasan mengapa belajar lewat terminal sangatlah penting. Tapi begitu kamu sudah mulai paham, kamu akan bersyukur punya aplikasi desktop karena aplikasinya menampilkan informasi projek dengan lebih mudah dipahami, sama seperti di situs Github.

Git sendiri tidaklah sama dengan program lainnya yang ada di komputermu. Kamu tidak akan melihat sebuah icon di desktopmu, tapi akan selalu tersedia untukmu dan kamu bisa mengaksesnya kapanpun kamu mau lewat terminal atau aplikasi desktop Git.

Pengaturan Git

Saat aplikasi Github Desktop (dan juga Git) sudah terpasang, buka terminalmu. Kamu dapat memastikan bahwa Git memang sudah terpasang dengan mengetik:

git --version

ini akan mengembalikan versi Git yang ada di komputermu dan tampak seperti ini:

git version 1.9.1

Selanjutnya, mengatur Git agar bisa menautkan pekerjaanmu kepadamu:

Atur namamu:

git config --global user.name "Nama Kamu"

Sekarang emailmu:

git config --global user.email "emailmu@contoh.com"
