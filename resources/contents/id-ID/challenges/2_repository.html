Buat repositori baru di komputermu.

Repositori

Sebuah repositori adalah kumpulan dari beberapa item yang berkaitan. Dalam kasus ini, saat menulis aplikasi, itu adalah kumpulan dari beberapa file yang berkaitan dengan projek aplikasinya. Kamu bisa membayangkannya seperti sebuah folder projek dengan semua file yang berkaitan didalamnya. Faktanya, itu sama saja seperti yang terlihat di komputermu. Kadang disebut juga sebagai "repos" untuk singkatnya.

Kamu memberi tahu Git yang mana projekmu dan Git akan mulai memantau segala perubahan yang terjadi di folder itu. Ini menjadikannya sebuah repositori Git: sebuah folder yang dipantau oleh Git. Git memantau saat ada file ditambahkan, dikurangi, atau bahkan sebuah huruf di sebuah file dirubah. Semua ini ditambah dengan siapa yang melakukannya dan kapan dilakukannya juga dipantau oleh Git. Dalam sebuah aplikasi, pemantauan perubahan seperti ini disebut kontrol versi.

Menggunakan Terminal

Terminal (atau Bash) adalah sebuah cara untuk memerintahkan komputermu melakukan hal dengan mengetikkan perintah daripada mengeklik aplikasi dengan menggunakan mouse milikmu. Kamu dapat mengubah nama file, membuka file, membuat folder baru, pindah ke lain direktori (folder) dan lain sebagainya hanya dengan mengetikkan perintah. Kamu bahkan bisa menggunakan pengedit teks untuk koding (seperti Vim) di dalam terminal dan tanpa harus meninggalkanny!

Selain memerintah komputermu, kamu juga bisa menggunakan program di Terminal yang mempunyai Antarmuka Baris Perintah (Command-Line Interface, CLI), yang artinya dapat dijalankan dengan perintah di terminal. Git adalah salah satunya. Bagian pertama dari perintahnya memberi tahu komputermu bahwa kamu sedang berinteraksi dengan Git. Lalu bagian selanjutnya adalah perintahnya dan berbagai macam pilihan yang kamu inginkan dari Git untuk bekerja.

Di Git-it kamu akan belajar beberapa aksi baris perintah yang dasar (sebagai tambahan dari perintah Git) untuk mengoperasikan komputer milikmu; semuanya dijelaskan dalam bagian-per-bagian.

Membuat Repositori

Kamu akan membuat sebuah folder projek baru dan menjadikannya sebagai repositori Git. Di tantangan ini kita akan menamai foldernya 'hello-world', tapi kamu boleh memberi nama lain jika kamu sudah punya folder dengan nama ini (karena sudah biasa menjadi nama projek pertama).

Saatnya buka terminal milikmu!

Di terminal, ketikkan perintah berikut, satu persatu setiap barisnya, lalu tekan enter.

Pertama, membuat folder baru:
Tip: mkdir kependekan dari make directory (buat direktori)

mkdir hello-world

Lalu masuk ke folder itu:
Tip: cd kependekan dari change directory (pindah direktori)

cd hello-world

Terakhir, beri tahu Git untuk memulai (mulai memantau) folder yang kamu sedang didalamnya ini:

git init

Perintah terakhir harusnya mengembalikan sebuah pesan seperti ini "Initialized empty Git repository". Perintah yang lainnya tidak mengembalikan pesan apapun.

Kamu berhasil! Jika kamu mau tahu dan benar-benar yakin itu sudah jadi repositori Git, maka ketikkan git status dan jika TIDAK mengembalikan pesan 'fatal: Not a git repository...', kamu Berhasil!
